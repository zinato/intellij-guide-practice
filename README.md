# Intellij Guide Pratice

> IntelliJ 단축키 정리와 사용법 연습하는 프로젝트입니다.

# Git

1. Git View On
   - Mac : Command + 9
   - Window, Linux: Alt + 9
2. Git Option Popup
   - Mac : Crtl + V
   - Window, Linux : Alt + `
3. Git History
   - Mac : Crtl + V => 4
   - Window, Linux : Alt + ` => 4
4. Branch
   - Mac : Crtl + V => 7
   - Window, Linux: Alt + ` => 7
5. Commit
   - Mac : Command + K
   - Window, Linux : Crtl + K
6. Push
   - Mac : Command + Shift + K
   - Window, Linux : Crtl + Shift + K
7. Pull
   1. intellij 에서 아직 지원하지 않음
      - Mac : Command + Shift + A => git pull
      - Window , Linux : Crtl + Shift + A => git pull
